# RC Airplane

### Used components:
Airplane:
- 1x ESP32 DevKit 1
- 1x nRF24L01+ PA/LNA Ext. Antenna
- 1x ESC 30A
- 1x Brushless DC Motor 2200kv
- 4x 9g Servo Motor
- 4x LED (1R, 1G, 2W)
- 4x Resistor 330
- 1x Polarized Capacitor 10uF
- 1x AMS1117-3.3V
- 1x UBEC 5V/3A Freewing
- 1x MPU6050
- 1x BMP280
- 1x ADS1115
- 1x 5200mAh 3S Lipo Battery

Control:
- 1x Arduino Nano
- 1x nRF24L01+ PA/LNA Ext. Antenna
